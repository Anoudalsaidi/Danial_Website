# 🌐 Danial Website

A modern and responsive personal website built using HTML, CSS, and JavaScript.

## 📌 Overview

Danial Website is a clean and user-friendly web project designed to showcase personal information, skills, projects, and contact details through an attractive and responsive interface.

The website focuses on providing a smooth user experience across different devices and screen sizes.

## ✨ Features

- 🎨 Modern and clean design
- 📱 Fully responsive layout
- 🚀 Fast loading performance
- 🧩 Structured and maintainable code
- 💼 Personal portfolio section
- 📞 Contact information section
- 🌐 Cross-browser compatibility

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Responsive Design
